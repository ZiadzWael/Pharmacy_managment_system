# Pharmacy Management System

A comprehensive Java-based pharmacy management application with Swing GUI for inventory control and order processing.

## Features
- **Inventory Management**: Add, remove, and track drugs/cosmetics
- **Categorization**: Drugs, Cosmetics, Other, Prescription-only items
- **Order Processing**: Real-time quantity validation and price calculation
- **Prescription Validation**: Checks for prescription-only medications
- **Capacity Control**: Set and monitor storage limits
- **Daily Sales Tracking**: Automatic calculation of daily revenue
- **Swing GUI**: User-friendly tabbed interface

## Categories
- **dr** - Regular drugs
- **co** - Cosmetics (20% markup)
- **ot** - Other medical items
- **not** - Prescription-only drugs
